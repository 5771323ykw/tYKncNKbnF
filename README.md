# 旅游信息管理系统 java595

## 项目概述

旅游信息管理系统是一款基于前后端分离架构的信息管理系统，旨在为旅游行业提供一站式的信息化解决方案。系统包含前台展示和后台管理两个部分，为用户提供旅游新闻、景区信息、美食信息、旅游线路等丰富的内容，同时为管理员提供便捷的管理功能。

## 技术栈

### 后端技术

- SpringBoot
- MyBatis

### 前端技术

- Vue
- Axios
- ElementUI
- JSP

### 开发环境

- Java 1.8
- Node.js
- MySQL
- Maven

## 功能模块

### 前台

- 展示旅游新闻
- 展示景区信息
- 展示美食信息
- 展示旅游线路
- 在线留言

### 后台

#### 管理员

- 账号管理
- 地区管理
- 景点信息管理
- 地方美食管理
- 旅游线路管理
- 订单信息管理
- 新闻管理
- 系统管理

#### 普通用户

- 旅游线路查询
- 订单信息查询

## 系统角色

- 管理员：负责系统的管理与维护
- 普通用户：浏览与查询旅游相关信息

## 运行环境

- 后端：Java 1.8+，MySQL 5.7+
- 前端：现代浏览器

## 部署步骤

1. 准备运行环境（Java、MySQL、Node.js等）
2. 导入数据库文件
3. 部署后端服务
4. 部署前端服务
5. 启动应用
